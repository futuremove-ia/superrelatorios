# 📦 Arquivo de Documentação

> **Leitura apenas** — Documentos obsoletos que foram substituídos por versões mais atuais.

---

## Propósito

Este diretório contém documentação histórica e obsoleta do projeto. Os arquivos aqui **não devem ser usados** como referência — eles servem apenas para contexto histórico e auditoria.

---

## Estrutura

```
ARCHIVED/
├── specs/          # Especificações e requisitos antigos
├── audits/         # Relatórios de auditoria
├── plans/          # Roadmaps e planos antigos
└── legacy/         # Documentos legado diversos
```

---

## ⚠️ Regra Importante

**NUNCA delete arquivos do ARCHIVED**. Eles são parte do histórico do projeto.

Se precisar referenciar algo aqui, prefira primeiro verificar se existe uma versão atualizada em:

- `docs/` — documentação pública
- `knowledge/` — base de conhecimento ativa

---

## Para Arquivar Novos Documentos

Quando um documento se tornar obsoleto:

1. Mova para a pasta `ARCHIVED/` apropriada
2. Adicione ao topo do arquivo: `# ⚠️ ARQUIVADO - Não usar`
3. Não modifique o conteúdo original

---

_Este diretório é mantido para referência histórica apenas._
