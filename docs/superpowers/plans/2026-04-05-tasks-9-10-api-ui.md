# Task 9-10: API Routes + UI Components

## Roles & Permissions Formalizados

### 1. Platform Admin (Dono da Plataforma SuperRelatórios)

| Atribuição           | Descrição                                                    |
| -------------------- | ------------------------------------------------------------ |
| **Responsabilidade** | Configurar e pagar pelos serviços de IA                      |
| **Decisões**         | Qual provider de embeddings usar (HF, OpenAI, Gemini, Local) |
| **Acesso**           | Configurações globais da plataforma                          |
| **Custo**            | Paga API de embeddings para todos os tenants                 |
| **Onde acessa**      | `Settings > AI Tab`                                          |

### 2. Tenant Admin (Admin da Empresa Cliente)

| Atribuição           | Descrição                                  |
| -------------------- | ------------------------------------------ |
| **Responsabilidade** | Gerenciar dados da empresa                 |
| **Decisões**         | Conectar/desconectar Google Drive/OneDrive |
| **Acesso**           | Dashboard, Reports, Settings do tenant     |
| **Onde acessa**      | `Settings > Conexões na Nuvem`             |

### 3. Tenant User (Usuário Final)

| Atribuição           | Descrição                               |
| -------------------- | --------------------------------------- |
| **Responsabilidade** | Usar o sistema para tomar decisões      |
| **Decisões**         | Perguntar ao AI Analyst                 |
| **Acesso**           | Dashboards, AI Analyst, Reports         |
| **Onde acessa**      | `AI Analyst`, `Métricas`, `Prioridades` |

## Fluxo de Dados

```
Platform Admin ──► Config Embeddings ──► Platform Config (Supabase)
     │                                                │
     │                   ┌────────────────────────────┘
     │                   ▼
Tenant Admin ──► Conectar Cloud ──► cloud_configs (per org)
     │
     │              ┌─────────────────────────────────┐
     │              ▼                                 ▼
Tenant User ──► AI Analyst ──► rag_documents ──► rag_chunks
```
